# Code of Conduct

This is a personal project I share with the community. I want it to be a welcoming space for anyone who finds it useful, whether you're building your first agent or shipping production AI.

## A few expectations

- **Be kind.** Assume good faith. Critique ideas, not people.
- **Be helpful.** If someone's stuck, help them — or point them somewhere that can.
- **Stay on topic.** Issues and PRs should be about the boilerplate.
- **Respect privacy.** Don't share other people's info without their consent.

## Not OK

Harassment, hate speech, sexualized comments, personal attacks, doxxing — anything that makes this project feel unsafe or unwelcoming.

## If something goes wrong

Email me at **francamacdowell@gmail.com**. I'll read it and handle it. That might mean a private conversation, removing comments, closing issues, or blocking accounts.

That's the whole thing. Use common sense, and let's keep this a friendly place to build.
